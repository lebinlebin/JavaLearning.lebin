offline 预估值分析
1. MAPE：评估模型拟合的能力。
	a. ZLIN loss属于回归loss的一种。MAPE用于评估模型的拟合能力。
2. Gini：评估模型对“头部消费者”的预测能力。
	a. 问：为什么Gini能够评估“头部”预测能力？
	答：Gini评价的是模型对于不同关注对的分辨能力，即“差异性”。而单列电商单次请求透出一般只有一个，所以需要重点评估模型对头部“关注对”分辨力。（类似于wuauc）
	b. 问：MAPE与Gini的区别？
	答: MAPE对尾部和头部pair对一视同仁，适合评价模型拟合能力；而gini能更好地契合真实线上指标。
3. MAP：目标检测领域的指标
